# Emotionalyst

### 당신의 감정을 실시간으로 분석해 보여드립니다 😎

## 기술 스택

- CoreML
- CreateML
- SoundAnalyst
- CoreAudio
- Swift
- SwiftUI
- MVVM

## 기능

- 사용자의 목소리를 실시간으로 받아서 MLModel에 분류된 감정을 보여줍니다.
